## Highlights (why post2)

### Fixes (headline)

- **`fix(connectors)` — symlink / reparse-point loop guard ([#1080](https://github.com/FabioLeitao/data-boar/pull/1080)):** Stops traversal dead-loops on cloud reparse points and symlink edge cases — **security / reliability** on real filesystem scans.
- **`fix(report)` — Excel sheet-name sanitize:** openpyxl rejects `/` in sheet titles; praise/controls sheet renamed to a safe title so reports complete on Windows operator paths.

### Also included in this build (not headline)

- **`feat(licensing)`** — tier / entitlement surfaces merged on `main` since post1.
- **`data-boar --demo`** ([#1113](https://github.com/FabioLeitao/data-boar/issues/1113) / [#1119](https://github.com/FabioLeitao/data-boar/pull/1119)) — zero-config turnkey demo for first-time `pip install` users.
- Operator help / docs sync (`QUICKSTART.md` Caminho 0, man page, dashboard help).
